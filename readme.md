# Node REST API

> Full CRUD REST API using Node.js with no framework

This is for learning/experimental purposes. For production project I would like to use Express

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
